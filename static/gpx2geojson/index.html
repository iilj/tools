gpx2geojson / gpx2kml

説明

GPX を GeoJSON または KML に変換します．JavaScript 単体で記述されています．

座標を適当に間引く設定が欲しかったため，変換機を自作しました．

出力の座標数が「座標数上限」を上回らないように座標を間引きます(*)．この値を 5000 未満程度に設定すると，地理院地図に読み込んだ際に軽快に動きます．

(*) 正確には，座標数上限に従って決まる「座標採用間隔」の値が N になるとき，座標が N 個に 1 個の割合で等間隔に採用されます．
