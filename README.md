iilj-tools
=====

## Description

雑多なツールを置いておく場所．

- 競プロ向け
    - wolfram2cpp for 競プロ
    - leetcode2cpp for 競プロ


## build

ツールごとに行う．

```sh
$ npm run build -fn=src/wolfram2cpp/rollup.config.js
```

```sh
$ npm run build -fn=src/leetcode2cpp/rollup.config.js
```


## serve

```sh
$ npm run serve
```


## test

```sh
$ npm run test
```
